# Food Menu System (Java Final Project)

## Overview

This project is a Java-based food ordering system that allows users to view menu items, place orders, view current orders, and complete a checkout process. The system was developed as part of a final project to demonstrate key programming concepts.

## Features

* View available food menu
* Add items to order with quantity
* View current orders
* Checkout and generate total receipt
* Menu-driven system for easy navigation

## Technologies Used

* Java
* Arrays
* Classes and Objects
* Methods and Loops

## How It Works

The system uses arrays to store menu items and customer orders. Users interact with the program through a console-based menu, where they can select different options to perform actions such as adding orders or checking out.

## Future Improvements

* Implement File I/O to store orders permanently
* Add a graphical user interface (GUI)
* Improve input validation and error handling
